This folder contains the release versions of 
 - libusb (with the Peter Batard's windows backend, updated to the commit 03e9371a460031dcb599a1cf504539667cf12f1c of the GIT repo)
 - pthread_win32 (which despite the name works also on 64bit Window versions)
 
The files in the AMD64 folder have been built for native 64bit Window applications;
the files in the X86 folder have been built for 32bit Window applications.

All the .lib and .dll files have been compiled with the /MD switch, without debug info.
